POLYMORPHISUM

polymorphisum is nothing but one object behaving many forms is known as polymorphisum

polymorphisum is achieved by using method overloading and method overriding

METHOD OVERLOADING :- Having same method name and different method signature is known as method overloading in python method overloading is not possible because it takes latest defind methos

METHOD OVERRIDING : have same method name and same method signature and the parent class is over readne with chaild class is known as method overriding
