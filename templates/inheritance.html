INHERITANCE

Inheritance is the process of inheriting properties from one class to another classs

There are maily five types of inheritance

SINGLE level INHERITANCE:- inheriting properties from parent class to chaild class

MULTILEVEL INHERITANCE :- inheriting properties from one chaild class to another chaild class

MULTILEVEL INHERITANCE :- inheriting the proverties from more than one parent class to one chaild class

HYERARICHAL INHERITANCE :- inheriting the properties from one parent class to more than one chaild class

HYBREAD INHERITANCE :- combination of morethan one tye of inheritance
